CREATE THE APIKEY AND SECRETKEY KEYS

DOWNLOAD THE LATEST VERSION OF BOOM_TRADER FROM THE TELEGRAM CHANNEL.

MUST BE INSTALLED. NET Core 3.1 runtime

1. Create a folder on the desktop "TRADING ROBOT-1" (the folder name can be any ), unpack the archive to the created folder.

2. Go to the "TRADING ROBOT-1" folder and run the file "BoomTrader_2.exe»

3. In the "connections" section. enter the received key in the "license key" field, in the API key and Sekret API fields, the received keys on binance, and click the "connect" button . the "logs" window will display a message about successful license verification.

4. Close, open trading robot.

5. Create on the desktop another 2 folders "TRADING ROBOT-2", "TRADING ROBOT-3". Copy all files from the folder "TRADING ROBOT 1" in the folder "TRADING ROBOT-2", "TRADING ROBOT-3".

Launch all 3 versions of the trading robot, and connect to the exchange.

When selecting contracts consider:

Deposit amount (if the Deposit is small, the allocated share may not be enough for a minimum purchase, exclude all expensive futures) with a Deposit of $ 300, you can trade all contracts.

With a$ 30 Deposit, exclude the 20 most expensive contracts.

Also consider the Funding fee on some Altos, it is very high and a long cycle will eat up the Deposit, for example, YFII Funding fee per day ~1.5% of the contract volume, and holding such a position for 3 days will reduce the profit from this cycle to ~0

Distribute futures contracts evenly across 3 bots. !!!NO MORE THAN 25 CONTRACTS!!!

For smooth operation of the bot, we recommend using the AMAZON remote desktop, choose London from locations, ping 1ms.

Learn more about settings.
