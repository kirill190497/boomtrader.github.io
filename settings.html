This tab contains the direct settings of the trading robot.

In the upper basket are all available instruments in the futures section, in the lower only those that will participate in trading in this particular bot. The fields below indicate how many items will be in each bucket, by default 4×4. (closed for editing)

1. «Position volume (in USDT)» by default, 50% of the volume is overwritten when connecting to the exchange. (closed for editing)

2. «Enter when there is a discrepancy, %» specify the difference between the baskets when the trading robot will enter the transaction

3. «Closed in profit, %» profit setting at which positions are closed

4. «Exchange leverage»

5. «Buy in addition,%» Indicate the step of additional purchase

6. «Number of purchases» we set the number of additional transactions, if the discrepancy still increases — thanks to this, it is possible to increase profits (additional purchases are made in both directions, when bot in — additional purchases average the position, when bot in + additional purchases increase the volume of positions)

7. «Stop loss, $» If you want to work with stop loss, check the box and enter the size in% in the field.

8. «Trailings» is activated upon reaching «Close on profit», by default 10%. (closed for editing)

START launches a trading robot (you must first specify the settings in the «Settings» block).

To stop a trading robot, press the STOP button.

RESET serves to reset the coefficients.

CLOSE closes all open positions.

Эта вкладка содержит настройки торгового робота.

В верхней корзине находятся все доступные инструменты в разделе фьючерсов, в нижней-только те, которые будут участвовать в торговле именно этим ботом. Поля ниже указывают, сколько элементов будет в каждом корзине, по умолчанию 4×4. (закрыто для редактирования)

1. «Объем позиции (в USDT) » по умолчанию 50% объема перезаписывается при подключении к бирже. (закрыто для редактирования)

2. «Вход при расхождении, %» укажите разницу между корзинами, когда когда заходить в сделку

3. « Закрыть при профите, %» настройка профита, при которой закрываются позиции, или включится трейлинг

4.«Биржевое плечо»

5. «Докупки,% » укажите шаг докупки

6. «Количество докупок » устанавливаем количество докупок, если расхождение увеличивается — благодаря этому можно увеличить прибыль, и быстрее закрыть сделку (дополнительные покупки делаются в обоих направлениях, когда бот в - докупки усредняют позицию, когда бот в + докупки увеличивают объем позиций)

7. «Стоп-лосс, $ » если вы хотите работать со стоп-лоссом, установите флажок и введите размер в %.

8. «Trailings» активируется при достижении «Закрыть при профите», длина по умолчанию 10%. (закрыто для редактирования)

START запускает торгового робота (сначала необходимо указать настройки в блоке "настройки").

Для остановки робота, нажмите кнопку Stop.

RESET для сброса коэффициентов.

CLOSE закрывает все открытые позиции.
